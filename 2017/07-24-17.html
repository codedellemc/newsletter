{code} Community

The ECS Community Edition is one of the projects that have been using the {code} Community Slack for communication and collaboration. ECS CE is a free-as-in-beer way to test out and play with a limited version of Dell EMC® Elastic Cloud Storage (ECS ™ ).

ECS provides support for industry-standard object APIs like Amazon S3, OpenStack Swift, EMC Atmos, and CAS . In addition, ECS also includes native file support with NFSv3 and support for Hadoop file system data, which allows organizations to create Big Data repositories on ECS that Hadoop analytics can consume and process. 

The project is taken care of by  Travis Wichert as the primary contributor, with great efforts also coming from Adrian Moreno , Magdy Salem , Ken Steinfeldt , and Jason Cwik .
