Automating Docker Swarm and REX-Ray installs in GCE with Ansible

Travis Rhoden shares his experience with Ansible  and REX-Ray. Rhoden explains that users should not be managing infrastructure and environments by hand. As an alternative, he provides a step-by-step guide on how to use Ansible for automation.

Rhoden begins by covering how to install an Ansible role or Puppet module, both of which can be used to configure this kind of automation. For the purposes of this guide, Rhoden walks us through using Ansible to deploy a REX-Ray enabled Docker Swarm in a Google Compute Engine. 

Check out the blog for more information. 

Give Mesos and external volumes a spin with playa-mesos

Clinton Kitson explains why Mesos is an important platform to consider if you are running containers. Kitson explains the reasons for adopting the Mesos platform, how to get started using  playa-mesos and takes us through the Quick Start guide found in the repo.
