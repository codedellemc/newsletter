Evaluate CoprHD Using Vagrant 

Christopher Banck, a hard-working EMC vSpecialist and Open Source community member  based in Germany, has been working on automated deployments and open source tools for a while. Banck says that "[Open Source] is fun and very approachable. If you find an issue or want to add a feature, just write it and make a pull request!" 

Within days of the CoprHD 2.4 release, Banck presented a really cool Vagrant setup that allows you set up quickly and efficiently. He has written a great blog on how to get started with it and we had the privilege of posting it on the EMC {code} Blog late last week. This is Banck's second contribution to the EMC {code} Community.

ScaleIO Integration with Mirantis OpenStack

EMC {code} team members, Magdy Salem and Adrian Moreno have been working hard on two plugins for ScaleIO and Mirantis Fuel. In the latest blog that explains these new plugins, Adrian explains: " The first plugin (i.e. ScaleIO plugin) is targeted at new deployments wishing to install OpenStack with an integrated and configured ScaleIO cluster. Whereas the second one (i.e., ScaleIO-Cinder plugin) is targeted at OpenStack deployments that want to leverage an already existent ScaleIO cluster." 

As you can see, these plugins are much needed in order to integrate ScaleIO with Mirantis Fuel. For more details see the blog post here.
