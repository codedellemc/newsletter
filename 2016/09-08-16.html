{code} Community

The ecs-dashboard project from Eric Caron is a toolbox designed to make it easy for anyone to create, metricize and visualize Dell EMC Elastic Cloud Storage (ECS) metrics such as top users, detailed object reports, and usage by namespace, bucket and node. The toolbox contains three subsystems: ECS Metadata Collector, ElasticSearch Metadata Cleaner and Kibana Emailer. All of which make the ecs-dashboard project very simple to setup and use.
