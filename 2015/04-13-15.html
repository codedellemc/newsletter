Introducing ViperPy and #DevHigh5 Winner, Chad Lung!

 

It's been a busy month for the {code} community, with six new #DevHigh5 projects posted on GitHub. ViperPy is one of them. #DevHigh5 winner Chad Lung describes the project in his recent blog post: 

 

"ViperPy is a Python library for interacting with the EMC ViPR API. ViperPy began originally as an easy way to hook up numerous projects that needed to interact with the ViPR product in order to perform certain metering and billing extraction and handle authentication tokens behind the scenes."  

 

Chad realized that this could be helpful for his team if he added support for additional APIs and other features. ViperPy offers a means for Python users to write their application, as an alternative to the Java SDK. Check out ViperPy and follow Chad on Github and Twitter!
