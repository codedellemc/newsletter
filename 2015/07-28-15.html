Top 25 EMC {code} projects  

 

In honor of the 25th newsletter issue, we're recapping the top 25 projects from the EMC {code} team, in no particular order:

  1. REX-ray: A guest based storage introspection tool that allows local visibility and management from cloud and storage platforms
  2. Vipruby: Ruby Gem for interacting with EMC's ViPR controller REST API
  3. {code} photobooth: A Photo Booth application that tweets and sends emails. Written in Node.js & Express.js using MongoDB, EMC ECS S3 API, Amazon SES API, Twitter API, and ConstantContact API. Showcased at EMC World 2015
  4. goair: A Go based multi-platform CLI tool for VMware vCloud Air services
  5. puppet-scaleio: A Puppet module for installing, and configuring ScaleIO data services components.
  6. arrowhead: Enabling Docker containers on top of hyper-converged machines with ScaleIO storage
  7. clue: Clue is a Go package that enables you to store persistent variables between application runs
  8. gotablethis: This Go package will allow you to format and print tables with data
  9. s3motion: Object Migration between S3 Compatible Storage Providers
  10. core2f: This project contains relevant steps, automation tools instructions, and eventually Docker containers to help minimize friction when deploying CoreOS to vSphere/vCloud
  11. How-to videos: EMC Code 101-level videos on DevOps concepts
  12. snake charmer: Collection of tools to create simple ways to test and use object services with Docker containers
  13. docker-ovftool: Docker-ovftool allows you to create a Docker image that includes ovftool
  14. socieidos: A collection of micro-services that actively watches multiple social media mediums for specific media and retains this data for real-time consumption and analytics use cases
  15. goxtremio: A package that provides XtremIO storage platform API bindings
  16. ecs-dockerswarm: Deploy EMC's ECS as a multi-node Docker container setup using Docker Compose into a Docker Swarm cluster created by Docker Machine.
  17. DevOps @ EMC World: EMC's first DevOps event at EMC World 
  18. VStriker: ViPR Data Services Throughput Analyzer
  19. goscaleio: package that provides API bindings for Go
  20. HeliosBurn: A REST Fault Injection Platform
  21. mosaicme: Micro service application that retrieves images from Twitter feeds, stores and process them to build a mosaic based on a selected picture.
  22. vagrant-puppet-scaleio: Vagrant files that prepare a Puppet Master and Agent nodes to configure ScaleIO data services using a Puppet module.
  23. dogged: A repository of Container Engine or Runtime related integration work for REX-Ray
  24. puppet-vnx: This module wraps Naviseccli into a Puppet module to simplify and dramatically increase the VNX configuration efficiency, especially for users who have multiple VNX arrays. 
  25. vagrantspice: The purpose of VagrantSpice is to simplify and standardize how the different machine providers are configured across Vagrant cloud providers.
