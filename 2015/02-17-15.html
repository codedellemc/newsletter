Introducing s3motion

 

Recently, the EMC {code} team launched s3motion. "Object" is common place in new applications and many vendors are offering this new generation of object-based storage. The one thing that has made object storage so successful is the lowest common denominator of S3.

 

S3motion is an open project to simplify the movement and migration between various Object-based storage systems that use the S3 API - such as EMC ViPR, EMC Atmos, AWS S3, RiakCS, vCloud Air Object Storage and others.

 

Read more about s3motion on the EMC {code} blog.

Meet #DevHigh5 Winner, Denis Jannot 

 

EMC Principal Systems Engineer (SE) Denis Jannot created two new projects, "Web Automation Center" and "Dockerized ScaleIO." Web Automation Center creates a learning environment to understand how to interact with multiple REST APIs. Dockerized ScaleIO simplifies how to create a small POC environment for testing ScaleIO. Both of these projects show how automation and modern technology (eg. Docker) can be applied to traditional data center environments to improve efficiency of IT operations.
