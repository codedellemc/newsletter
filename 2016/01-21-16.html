First Docker Meetup in Boston of 2016

This week the Docker Boston Meetup was hosted at VMTurbo's amazing offices in downtown Boston. To start off the evening, NVBOTS' Forrest Pieper and Areth Foster-Webster, showed off how to use Resin.io to push out Docker containers to their large fleet of 3D printers located at various customer sites around the globe. They also printed a large number of Docker whale keychains and handed them out to the audience. A very nice touch!



EMC {code}'s Jonas Rosland also presented, and did a full live demonstration of how to set up Mesos and Marathon that allows for persistent applications such as PostgreSQL, Redis and Cassandra.

Volume Plugins with Docker Toolbox and Boot2Docker

In our latest blog, Clint Kitson takes us through the process of running Docker Toolbox and Volume Plugins. Docker Toolbox is described as a set of tools that allows you to run a Docker environment on your computer. Kitson does a great job breaking down the blog into three sections: installing, deploying and using the Docker Toolbox. 

For more information on Boot2Docker and how to utilize the Docker Toolbox visit the blog.
