Have you ever wanted to be seen as a Gopher? Thanks to the great http://gopherize.me project by {code} Catalyst Ashley McNamara and web app developer Mat Ryer, you can! It's fun, silly  and seriously cute. After making his own Gopherized self, Jonas Rosland reached out to Ashley to find out more and here is what she had to say: 
 
JR: What sparked the idea?
AM: It all started on Twitter. I've been doing a lot of Gopher logos for user groups and conferences around the world. And as a joke, I used some spare parts and made an avatar for Brian Ketelsen. B efore I knew it, I was getting requests for more avatars.  Todd Rafferty tweeted, "We should just make an avatar generator" and then not a minute later I get a DM from Mat Ryer saying, "We can have this up in a few hours." It was up and running that night.
 
JR: How long did it take?
AM: The initial build took about 4 hours, including design work because we used spare parts from logos already made. V2 took about a week because it involved new designs. Every item on gopherize.me is a community request, which was a beautiful thing.
 
JR: What tools did you use?
AM: Photoshop, Adobe Illustrator, Go, Some HTML/CSS, Javascript, and Google Cloud Storage. We collaborated online using Twitter and Slack.
 
JR: Are you working on something new?
AM: We're working on giving users the ability to select custom hair colors and there's still a fair amount of design work that needs to be done. We recently linked Gopherize Me and Zazzle so you can now order personalized Gopher swag!

JR: Anything special that you'd like to mention?
AM: To quote my friend Matt Broberg, "Culture is such a fundamental part of open source projects. The Go programming language has built a space where people can step up and be really fun with design in a way I haven't seen in other languages or projects." I cannot agree with this more. The Go community is one of the best I've experienced and if it weren't for the community this project wouldn't exist.

REX-Ray 0.8 Boasts Official Docker 1.13 Plugin Support along with DigitalOcean, S3FUSE, FittedCloud, Azure and Google Storage Platforms

It's here! The REX-Ray v0.8.0 release - our most feature-rich release yet. Here's a few highlights among the added features: 
  • Two community-contributed drivers were released
  • Over 10,000 new lines of code were added with fewer than 1,000 deletions
REX-Ray continues to be the leading open source container storage orchestration engine. Open source, being the keyword, allows anyone to contribute storage drivers that provide effortless integration to Docker, Mesos, and Kubernetes.
