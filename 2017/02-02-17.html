Deep Dive into Docker 1.13 Storage Plugins and Propagated Mounts

The Managed Plugin System is the newest supported feature in Docker 1.13. The ultimate goal and use of a Managed Plugin is to take advantage of a plugin's extensibility quickly and easily.  Today we have a tech-preview of REX-Ray using the Managed Plugin System.

Read more in the blog by  Kenny, which includes a deep-dive into volume mounts.
