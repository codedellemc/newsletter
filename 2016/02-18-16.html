New DevHigh5 Project Highlight - PyXtrem

Last week we introduced five new DevHigh5 Open Source projects. Over the next five weeks, we will share a brief overview of these new projects. 

PyXtrem is a project focused on creating a library of python modules in order to simply the usage of XtremIO REST API. With this library, users can automate workflows related to XtremIO management functions such as adding/deleting an XtremIO volume, initiator  or initiator group, consistency groups, creating snapshots on volumes or consistency groups, refreshing a snapshot and many more.

To view the library and learn more about this project, take a look at the GitHub ReadMe file for more information.
