Isilon SDK Launches Into Open Source
 
As the open source initiative for EMC we love the opportunity to guide other teams into establishing their own open source projects. We walk these teams through this transition from closed source to open source and for Isilon SDK it has finally become a reality. 

With this SDK you can now add bindings to your applications in order to access the OneFS API on an Isilon cluster. This will make it much easier to programmatically manage your Isilon cluster and paves the way into treating your infrastructure as code!

For more head to the repo or check out the documentation here.
