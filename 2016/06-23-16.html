What's a community without contribution?

 
Did you hear that on Monday, June 20th we announced the release of RackHD Machine Driver for Docker? This particular release was due to the contribution of the original project by the RackHD team and Kendrick Coleman. 

The new enhancements featured are: 
  • Enables lifecycle management of a physical server for Docker solutions
  • Completely agnostic of server vendors, enabling a common orchestration capability across the data center
  • Enables extra integrations and operating systems through customizable workflows
  • Leverages Docker containers for simplified operational experience
  • Can also be used to provision other application platforms
  • Enables private cloud bare-metal deployment to simplify the container platform experience through integration with Rancher 1.0
It's releases like this that continue to show the importance of community and contribution within open source. It emphasizes  the power found in the hands of many that provides an ability to release often, make improvements and fixes quickly for open source projects such as RackHD. 

For more information and to get started head to our github page for featured projects including RackHD or head to the Community.

DockerCon 2016 enticing the audience

The atmosphere at DockerCon was electrifying this year. It was exciting to see 4,000 people in Seattle interested in Docker. Over the past few days, we had brilliant conversations with many people, most of whom were trying to figure out the best way to make use of Docker in their environments.


There were a few key themes at DockerCon 2016. First was Docker at scale where Docker 1.12's SwarmKit brings native clustering functionality into Docker. This will certainly simplify the process of building secure and scalable docker clusters. The second theme was Docker for developers. Docker kicked off the public beta of Docker for Windows & Mac. Being able to run docker natively on these platforms will help simplify the creation and management of development environments, as well as unlock many new use-cases for Docker.  Lastly, the second day's keynote focused on Docker in the Enterprise. Docker is making a big push with Docker Data Center to make Docker more accessible for Enterprise customers looking to take advantage of containers.
