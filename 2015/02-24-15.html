Meet #DevHigh5 Winner, Chris Wahl

EMC partner and virtualization guru, Chris Wahl, has achieved #DevHigh5 status with his project, NSX Tier Builder. This project was created to quickly stand up and tear down virtual networking environments with VMware NSX. NSX Tier Builder was created with a few use cases in mind: delivering a better understanding of the NSX API from a PowerShell coder's perspective, provisioning and removing network tiers for dev work, and combining vRealize Orchestration input/outputs with a generic framework to create a desired outcome.
